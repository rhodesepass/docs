
# 白银电子通行证使用说明书

::: tip 适用版本
- PCB版本：v0.3.1、v0.4、v0.6
- 软件版本：a1.4、a2.1
:::

## 1. 系统配置

### 1.1 配置默认参数说明

系统主要配置参数包括：

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| Brightness | 屏幕亮度 | 50% |
| Switch Interval | 轮播切换间隔 | 3分钟 |
| Switch Mode | 轮播切换模式 | Sequence（顺序） |
| USB Mode | USB连接模式 | Download（下载模式） |
| Low Battery Trip | 低电量提醒 | 开启 |
| No Intro Block | 跳过开机动画 | 关闭 |
| No Overlay Block | 禁用Overlay层 | 关闭 |

### 1.2 验证方法

开机后，系统会自动播放开机动画，然后进入干员展示界面。验证以下功能是否正常：
- 按键响应（按键1-4是否正常工作）
- 干员切换（是否能正常切换不同干员）
- 菜单操作（是否能进入设置菜单并调整参数）
- 扩列图显示（是否能正常显示和隐藏扩列图）

## 2. 功能模块详解

### 2.1 设备部件定义

| 部件 | 说明 |
|------|------|
| 按键1 | 切换上一个角色 / 菜单向上 |
| 按键2 | 切换下一个角色 / 菜单向下 |
| 按键3 | 展示扩列图 / 菜单确认 |
| 按键4 | 打开设置菜单 / 返回 |
| 按键5 | 预留 |
| SD卡槽 | 插入SD卡进行素材传输 |
| 开关 | 上推开机，下推关机（v0.6版为轻触按钮） |
| 充电口 | Type-C 充电接口 |
| 调试/拓展排针 | 用于调试和功能扩展 |

**图1：设备总览图** - 展示了电子通行证的各个部件及其功能

### 2.2 开机与关机

#### 开机操作
1. 上推开关至"ON"位置（v0.6版为轻触按钮）
2. 等待开机动画播放结束
3. 系统自动进入第1个干员展示界面

::: info 注意
保存关机时角色的功能将在未来软件更新中加入
:::

#### 关机操作
1. 长按按键4进入主菜单（v0.6版可长按关机）
2. 按动按键1/2使光标移动到"Shutdown"
3. 按下按键3确认
4. 等待系统关机完成后，下推开关至"OFF"位置

### 2.3 素材切换与播放调整

#### 2.3.1 素材切换

| 操作 | 按键 | 预期结果 |
|------|------|----------|
| 切换上一个角色 | 按下按键1 | 切换到上一个干员，显示对应角色动画 |
| 切换下一个角色 | 按下按键2 | 切换到下一个干员，显示对应角色动画 |

#### 2.3.2 扩列图展示

- 展示/隐藏扩列图：按下**按键3**

#### 2.3.3 播放亮度调整

1. 按下**按键4**进入主菜单
2. 按动按键1/2使光标上/下移动，对准 `Brightness`
3. 按下**按键3**确认
4. 按动按键1/2增加/减少亮度
5. 按下**按键3/4**退出

#### 2.3.4 轮播切换间隔调整

1. 按下**按键4**进入主菜单
2. 按动按键1/2使光标移动到 `Switch Interval`
3. 按下**按键3**进入
4. 按动按键1/2调整切换时间（可选：1/3/5/10分钟）
5. 按下**按键3**保存设置

#### 2.3.5 轮播切换模式调整

1. 按下**按键4**进入主菜单
2. 按动按键1/2使光标移动到 `Switch Mode`
3. 按下**按键3**进入
4. 按动按键1/2调整切换模式：
   - `Sequence` - 顺序
   - `Random` - 随机
   - `Manual` - 保持
5. 按下**按键3**保存

### 2.4 干员列表查看

1. 按下**按键4**进入主菜单
2. 按动按键1/2使光标移动到 `Operator List`
3. 按下**按键3**进入干员列表界面
4. 按动按键1/2上下滚动查看干员列表
5. 按下**按键3/4**退出干员列表界面

### 2.5 素材管理与传输

#### 2.5.1 使用内部存储传输

1. 按下**按键4**进入主菜单
2. 按动按键1/2使光标移动到 `USB Download`
3. 按下**按键3**确认进入USB下载模式
4. 使用Type-C数据线连接电脑
5. 电脑会识别到一个名为"Electric Pass"的存储设备
6. 将素材拖入 `\Electric Pass\assets` 文件夹中
7. 传输完成后，按下任意键重启设备
8. 重启后可拔出数据线

#### 2.5.2 使用SD卡传输

1. 下载教程并根据教程操作：
   - 链接：[百度网盘](https://pan.baidu.com/s/1m9FesnSgNn2ZP_sy4TnR7Q?pwd=2wk9)
   - 提取码：`2wk9`

2. 准备工具：
   - 软件：DiskGenius
   - 硬件：SD卡（最大支持32GB）

3. 按照教程将素材复制到SD卡指定目录
4. 将SD卡插入电子通行证的SD卡槽
5. 系统会自动识别并加载SD卡中的素材

### 2.6 素材制作

1. 下载制作器：
   - 链接：[百度网盘](https://pan.baidu.com/s/1B8EU96yhAKkfm191nf2MXg?pwd=yx8i)
   - 提取码：`yx8i`

2. 准备素材：
   - 角色视频（支持常用视频格式）
   - 通行证样式蒙版（可选）
   - 过场展示图片（可选）
   - 过场动画（可选）

3. 打开 `epass_flasher.exe` 并根据提示进行操作
